주요 기능

🔓

원터치 슬라이딩 도어 제어

블루투스 LE를 통해 스마트 슬라이딩 도어를 원터치로 열고 닫을 수 있습니다. 복잡한 설정 없이 간편하게 사용하세요.

📊

실시간 상태 모니터링

슬라이딩 도어의 현재 상태, 전원 상태, 연결 상태를 실시간으로 확인할 수 있습니다.

👥

다중 사용자 관리

가족이나 직원들과 함께 사용할 수 있습니다. 모든 사용자가 동일한 권한으로 슬라이딩 도어를 제어할 수 있습니다.

📈

사용 패턴 분석

슬라이딩 도어 사용 패턴을 분석하여 에너지 효율성을 높이고 최적화 방안을 제안합니다.

🔔

스마트 알림

슬라이딩 도어 상태 변경 시 즉시 알림을 받을 수 있습니다. 언제 어디서나 슬라이딩 도어 상황을 파악하세요.

⚙️

도어 캘리브레이션

설치 마법사를 통해 동작 속도, 열림 시간, 끼임 감지 등을 설정할 수 있습니다. 슬라이딩 도어에 최적화된 설정을 제공합니다.

사용 방법

1

앱 설치

App Store 또는 Google Play에서 SmartDoors 앱을 다운로드하고 설치합니다.

2

슬라이딩 도어 연결

블루투스를 통해 스마트 슬라이딩 도어를 앱에 연결합니다.

3

즉시 사용

연결이 완료되면 바로 슬라이딩 도어를 제어할 수 있습니다.
