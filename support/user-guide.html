SmartDoors 사용 가이드

스마트 슬라이딩 도어 앱의 모든 기능을 쉽게 사용하는 방법

📱 앱 시작하기

1. 앱 설치 및 실행

  1. App Store에서 SmartDoors 앱을 다운로드합니다.
  2. 앱을 실행하고 블루투스 권한을 허용합니다.
  3. 위치 권한을 허용합니다 (블루투스 스캔에 필요).
💡 팁: 처음 실행 시 모든 권한을 허용하면 더 원활한 사용이 가능합니다.

🔗 슬라이딩 도어 연결하기

2. 슬라이딩 도어 검색 및 연결

  1. 홈 화면에서 "슬라이딩 도어 추가" 버튼을 탭합니다.
  2. 블루투스가 켜져 있는지 확인합니다.
  3. 슬라이딩 도어의 전원이 연결되어 있는지 확인합니다.
  4. 다른 기기가 연결되어 있지 않은 상태(연결 대기 상태)인지 확인합니다.
  5. 검색된 슬라이딩 도어 목록에서 연결할 도어를 선택합니다.
  6. 연결이 완료되면 슬라이딩 도어 이름을 설정합니다.
⚠️ 주의: 슬라이딩 도어와의 거리가 5m 이내여야 안정적인 연결이 가능합니다.

🚪 슬라이딩 도어 제어하기

3. 기본 제어 기능

🔓 열기

슬라이딩 도어 카드에서 "열기" 버튼을 탭하여 도어를 엽니다.

🔒 닫기

슬라이딩 도어 카드에서 "닫기" 버튼을 탭하여 도어를 닫습니다.

📊 상태 확인

슬라이딩 도어의 현재 상태, 전원 상태, 연결 상태를 실시간으로 확인합니다.

⚙️ 설정

슬라이딩 도어별 개별 설정을 통해 사용자 정의가 가능합니다.

👥 도어 관리

4. 다중 도어 설정

  1. 홈 → + 로 이동합니다.
  2. "기기 추가" 메뉴를 탭합니다.
  3. 주변 기기를 검색하고 화면 지시를 따릅니다.
  4. 슬라이딩 도어 정보를 입력합니다.
  5. 각 슬라이딩 도어별 설정을 구성합니다.
  6. 저장을 탭합니다.
💡 팁: 모든 사용자는 동일한 권한을 가지며, 슬라이딩 도어 제어가 가능합니다.

📈 사용 패턴 분석

5. 에너지 효율성 모니터링

  1. 홈 화면에서 "사용 패턴" 메뉴로 이동합니다.
  2. 일별/주별/월별 사용 통계를 확인합니다.
  3. 에너지 소비 패턴을 분석합니다.
  4. 최적화 제안을 확인합니다.

🔧 고급 기능

6. 추가 기능들

🔔 알림 설정

슬라이딩 도어 상태 변경 시 푸시 알림을 받을 수 있습니다.

📱 위젯

홈 화면 위젯을 통해 빠른 슬라이딩 도어 제어가 가능합니다.

🔍 진단

슬라이딩 도어 문제 발생 시 자동 진단 기능을 사용할 수 있습니다.

⚙️ 도어 캘리브레이션

7. 설치 마법사 설정

  1. 기기 → 설정 → 도어 캘리브레이션으로 이동합니다.
  2. 설치 마법사를 실행합니다.
  3. 동작 속도, 열림 시간, 끼임 감지 등의 설정을 조정합니다.
  4. 변경사항을 저장합니다.
⚠️ 주의: 도어 캘리브레이션 설정 변경 시 비밀번호 입력이 필요합니다.

추가 도움이 필요하시면 문의하기를 이용해주세요.
