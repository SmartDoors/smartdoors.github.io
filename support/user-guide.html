SmartDoors 사용 가이드

최신 SmartDoors 앱은 YG-ADB100의 BLE 제어, 기기 등록 인증, WiFi 설정, OTA 업데이트, 진단과 에너지 데이터를 한 화면 흐름으로 관리합니다.

앱 설치

  1. 메인 페이지의 앱 다운로드 섹션에서 iOS 또는 Android 설치 경로를 선택합니다.
  2. Android 직접 설치 시 알 수 없는 앱 설치 허용이 필요할 수 있습니다.
  3. 앱 첫 실행 시 블루투스와 위치 권한을 허용합니다. Android에서는 BLE 검색을 위해 위치 권한이 필요할 수 있습니다.
앱 다운로드 버튼은 메인 페이지에서 iOS와 Android를 함께 제공합니다. 메인 앱 다운로드 섹션으로 이동

기기 등록과 BLE 인증

  1. 홈 화면에서 기기 추가를 선택합니다.
  2. YG-ADB100의 전원이 켜져 있고 다른 휴대전화에 연결되어 있지 않은지 확인합니다.
  3. 검색된 BLE 기기를 선택합니다.
  4. 기기 등록 단계에서 패스워드 인증을 진행합니다. 인증에 성공해야 해당 휴대전화가 도어에 등록됩니다.
  5. 도어 별칭과 설치 위치를 입력해 여러 도어를 구분합니다.
패스워드는 도어 등록과 설정 변경에 사용됩니다. 변경 후에는 다음 연결부터 새 패스워드가 적용됩니다.

기본 제어

열기/닫기

도어 카드에서 현재 연결 상태를 확인한 뒤 열기, 닫기, 정지 제어를 수행합니다.

상태 확인

도어 위치, 동작 상태, BLE 연결 상태, 전원 및 최근 수신 정보를 확인합니다.

장소 관리

여러 도어를 장소와 그룹 단위로 정리해 반복 사용 환경을 빠르게 찾습니다.

별칭 관리

앱 표시 이름과 도어 장치 이름을 구분해 현장 관리에 맞게 정리합니다.

설정과 유지보수

도어 캘리브레이션

설치 마법사에서 도어 이동 경로를 학습합니다. 진행 중에는 문 움직임을 방해하지 마세요.

WiFi 설정

도어의 공유기 WiFi 연결 또는 AP 모드 정보를 확인하고 OTA 접근에 필요한 네트워크를 설정합니다.

펌웨어 / OTA

도어 웹서버의 업데이트 페이지를 열어 펌웨어와 파일시스템 이미지를 업로드합니다.

기기 패스워드

설정 변경 권한을 보호하기 위해 현재 패스워드 확인 후 새 패스워드를 저장합니다.

진단과 에너지 데이터

진단연결된 도어에서 실시간 진단 데이터를 요청해 이상 상태를 확인합니다.
에너지총 에너지, 세션 에너지, 최대/평균 전력, 동작 횟수, 총 동작 시간을 확인합니다.
사용 패턴일별/월별 사용량과 에너지 흐름을 확인해 유지보수 판단에 활용합니다.
OTA 정보펌웨어 버전, 모델 UUID, MAC 주소, 빌드 정보를 설정 화면에서 확인합니다.

연결 문제가 있으면 연결 가이드와 문제 해결을 확인하세요.
