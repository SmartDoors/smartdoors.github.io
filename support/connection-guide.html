연결 가이드

SmartDoors 앱과 슬라이딩 도어를 연결하는 방법

📋 사전 준비사항

1. 호환성 확인

SmartDoors 앱은 다음 요구사항을 충족해야 합니다:

📱 기기

iPhone/iPad/Android
iOS 13.0 이상
Android 10.0 이상

🔗 블루투스

Bluetooth LE
5.0 이상

📍 위치

위치 권한
필요

2. 슬라이딩 도어 준비

연결할 슬라이딩 도어가 다음 조건을 만족하는지 확인하세요:

  • 슬라이딩 도어가 전원 연결이 되어 있음
  • 블루투스 LE 기능 지원
  • 다른 기기에 연결되어 있지 않음 (연결 대기 상태)
  • 앱에서 5m 이내에 위치
⚠️ 주의: 슬라이딩 도어가 다른 기기에 연결되어 있으면 SmartDoors 앱에서 검색되지 않습니다. 기존 연결을 해제한 후 다시 시도하세요.

🔗 연결 과정

3. 앱 실행 및 권한 설정

SmartDoors 앱을 실행하고 필요한 권한을 허용합니다:

  • SmartDoors 앱 실행
  • 블루투스 권한 요청 시 "허용" 선택
  • 위치 권한 요청 시 "앱 사용 중에만" 또는 "항상" 선택
💡 팁: 처음 실행 시 모든 권한을 허용하면 더 원활한 연결이 가능합니다.

4. 슬라이딩 도어 연결 상태 확인

슬라이딩 도어가 다른 기기에 연결되어 있지 않은지 확인합니다:

  • 기존에 연결된 다른 앱이나 기기가 있는지 확인
  • 다른 기기에 연결되어 있다면 해당 연결을 해제
  • 슬라이딩 도어의 전원이 켜져 있는지 확인
  • 슬라이딩 도어가 정상 작동 상태인지 확인
💡 팁: 슬라이딩 도어는 별도의 페어링 모드가 없습니다. 다른 기기에 연결되어 있지 않으면 자동으로 연결 대기 상태가 되어 검색됩니다.

5. 앱에서 슬라이딩 도어 검색

앱에서 슬라이딩 도어를 검색합니다:

  • 홈 화면에서 "+" 버튼 탭
  • "기기 추가" 메뉴 선택
  • 앱이 주변 블루투스 기기를 스캔하기 시작
  • 검색된 슬라이딩 도어 목록에서 연결할 도어 선택
💡 팁: 슬라이딩 도어가 검색되지 않으면 도어와의 거리를 확인하고 다른 기기에 연결되어 있지 않은지 확인하세요.

6. 슬라이딩 도어 연결 및 설정

슬라이딩 도어를 선택한 후 연결을 완료합니다:

  • 검색된 슬라이딩 도어 목록에서 연결할 도어 선택
  • 연결 진행 상황 확인
  • 연결 완료 후 슬라이딩 도어 이름 설정
  • 슬라이딩 도어 위치 정보 입력 (선택사항)
  • 연결 완료 확인
✅ 성공: 연결이 완료되면 홈 화면에 슬라이딩 도어 카드가 표시됩니다.

🔧 연결 후 설정

7. 슬라이딩 도어 테스트

연결된 슬라이딩 도어가 정상적으로 작동하는지 테스트합니다:

  • 슬라이딩 도어 카드에서 "열기" 버튼 탭
  • 슬라이딩 도어가 실제로 열리는지 확인
  • "닫기" 버튼으로 슬라이딩 도어 닫기 테스트
  • 상태 표시가 정확한지 확인

8. 추가 설정 (선택사항)

슬라이딩 도어 사용을 위한 추가 설정을 구성합니다:

  • 슬라이딩 도어별 개별 설정
  • 알림 설정
  • 도어 캘리브레이션 설정
  • 동작 속도 및 열림 시간 설정

🚨 문제 해결

연결 문제 해결

연결 중 문제가 발생한 경우 다음을 시도해보세요:

  • 슬라이딩 도어와의 거리 확인 (5m 이내)
  • 슬라이딩 도어 전원 상태 확인
  • 다른 기기에 연결되어 있는지 확인
  • 다른 블루투스 기기 연결 해제
  • 앱 재시작
  • 기기 재부팅
⚠️ 문제 지속 시: 문제 해결 가이드를 참고하거나 문의하기를 이용해주세요.

📚 추가 정보

지원되는 슬라이딩 도어

SmartDoors는 다음 블루투스 LE 슬라이딩 도어와 호환됩니다:

  • 블루투스 LE 기능을 지원하는 슬라이딩 도어
  • 자동 개폐 기능이 있는 슬라이딩 도어
  • 전원 연결 방식의 슬라이딩 도어
  • 끼임 감지 기능이 있는 슬라이딩 도어
💡 참고: 특정 슬라이딩 도어 모델의 호환성은 제조사에 문의하거나 앱 내 호환성 목록을 확인하세요.

연결에 문제가 있으시면 문제 해결 가이드를 참고하세요.
