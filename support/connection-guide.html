연결 가이드

YG-ADB100은 BLE 연결로 기본 제어와 설정을 수행하고, WiFi 또는 AP 네트워크를 통해 도어 웹서버와 OTA 페이지에 접근합니다.

사전 확인

휴대전화

iOS 13 이상 또는 Android 10 이상, 블루투스 활성화, 앱 권한 허용이 필요합니다.

도어 상태

전원이 연결되어 있고 다른 휴대전화와 BLE 연결 중이 아니어야 합니다.

거리

처음 등록할 때는 도어와 가까운 거리에서 진행하세요. 벽이나 금속 구조물은 신호를 약하게 만들 수 있습니다.

BLE 등록 절차

  1. 앱 홈에서 기기 추가를 선택합니다.
  2. 주변 BLE 기기 검색이 시작되면 YG-ADB100 또는 현장 도어 이름을 선택합니다.
  3. 기기 등록 버튼을 눌러 BLE 연결과 패스워드 인증을 진행합니다.
  4. 인증에 성공하면 도어가 앱에 저장되고 홈 화면에 카드로 표시됩니다.
  5. 연결 후 앱은 기기 정보, WiFi 상태, 진단, 에너지 데이터를 순차적으로 요청합니다.
검색되지 않으면 도어 전원, 다른 휴대전화 연결 여부, OS 권한, BLE 활성화 상태를 먼저 확인하세요.

WiFi / AP 연결

공유기 WiFi도어가 현장 공유기에 연결되면 앱은 해당 IP로 도어 웹서버에 접근합니다.
AP 모드공유기 연결이 없을 때 도어 자체 AP IP를 사용해 설정 페이지에 접근할 수 있습니다.
OTA 접근설정의 펌웨어/OTA 메뉴는 도어 웹서버의 /update 페이지를 앱 안에서 엽니다.
시스템 정보도어 장치 정보 조회를 통해 펌웨어 버전, 모델 UUID, MAC 주소, 빌드 정보를 가져옵니다.

연결 후 확인

  1. 도어 카드에서 연결 상태가 연결됨으로 표시되는지 확인합니다.
  2. 열기/닫기 제어가 실제 도어 움직임과 일치하는지 확인합니다.
  3. 설정 화면에서 펌웨어 버전과 모델 UUID가 표시되는지 확인합니다.
  4. 캘리브레이션이 필요한 신규 설치 현장은 설치 마법사를 실행합니다.

연결 실패가 반복되면 문제 해결 가이드를 확인하세요.
