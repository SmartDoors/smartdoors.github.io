Troubleshooting

문제 해결

BLE 검색, 도어 제어, WiFi/OTA, 캘리브레이션, 진단 데이터 문제를 최신 앱 동작 기준으로 점검합니다.

BLE 검색 또는 등록 실패

확인할 것

  • 휴대전화 블루투스가 켜져 있는지 확인합니다.
  • 앱의 블루투스/위치 권한이 허용되어 있는지 확인합니다.
  • 도어가 다른 휴대전화에 이미 연결되어 있지 않은지 확인합니다.
  • 도어 전원을 껐다 켠 뒤 가까운 거리에서 다시 검색합니다.
  • 패스워드 인증 실패 시 현재 기기 패스워드를 다시 확인합니다.

제어가 되지 않거나 상태가 맞지 않음

  1. 도어 카드의 BLE 연결 상태를 확인합니다.
  2. 도어 전원과 장애물 여부를 확인합니다.
  3. 앱을 완전히 종료한 뒤 다시 실행해 자동 재연결을 시도합니다.
  4. 기기 상세 또는 설정에서 진단 데이터를 요청합니다.
  5. 새 설치 현장이면 캘리브레이션을 다시 실행합니다.
캘리브레이션 중에는 도어가 완전히 열리고 닫히며 이동 경로를 학습합니다. 움직임을 방해하면 실패할 수 있습니다.

WiFi / OTA 페이지가 열리지 않음

IP 없음도어가 공유기 WiFi 또는 AP 모드에 연결되어 있는지 확인합니다.
같은 네트워크휴대전화와 도어가 같은 WiFi 대역에 있어야 도어 웹서버에 접근할 수 있습니다.
AP 모드공유기 연결이 어렵다면 도어 AP에 직접 연결한 뒤 OTA 페이지를 다시 엽니다.
업데이트 중 주의펌웨어 또는 파일시스템 업로드 중에는 전원을 끄지 마세요.

Android 앱 설치 문제

  1. 메인 페이지의 앱 다운로드 섹션에서 Android 앱을 다시 다운로드합니다.
  2. 브라우저 또는 파일 관리자에 대해 알 수 없는 앱 설치 허용을 켭니다.
  3. 기존 앱 위에 업데이트 설치가 실패하면 기존 앱을 백업 후 제거하고 다시 설치합니다.
  4. 설치 파일이 오래된 것 같으면 메인 페이지에서 최신 버튼을 다시 눌러 받습니다.
