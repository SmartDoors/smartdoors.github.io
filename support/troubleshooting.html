문제 해결 가이드

SmartDoors 앱 사용 중 발생하는 문제들을 해결하는 방법

🔗 연결 문제

슬라이딩 도어가 검색되지 않습니다

가능한 원인:

  • 블루투스가 비활성화되어 있음
  • 슬라이딩 도어가 다른 기기에 연결되어 있음
  • 슬라이딩 도어와의 거리가 너무 멀음
  • 슬라이딩 도어 전원이 꺼져 있음

해결 방법:

  1. 설정 → 블루투스에서 블루투스가 켜져 있는지 확인
  2. 슬라이딩 도어가 다른 기기에 연결되어 있는지 확인하고 연결 해제
  3. 슬라이딩 도어와 5m 이내에서 다시 시도
  4. 슬라이딩 도어 전원 상태 확인
  5. 앱을 완전히 종료하고 다시 실행

슬라이딩 도어 연결이 자주 끊어집니다

가능한 원인:

  • 슬라이딩 도어와의 거리가 멀어짐
  • 다른 블루투스 기기와의 간섭
  • 슬라이딩 도어 전원 문제
  • iOS 블루투스 설정 문제

해결 방법:

  1. 슬라이딩 도어 근처에서 사용
  2. 다른 블루투스 기기 연결 해제
  3. 슬라이딩 도어 전원 상태 확인
  4. 설정 → 블루투스 → 슬라이딩 도어 → "이 기기 잊기" 후 재연결

🚪 제어 문제

슬라이딩 도어가 열리지 않습니다

가능한 원인:

  • 슬라이딩 도어가 이미 열려있음
  • 슬라이딩 도어 전원 문제
  • 슬라이딩 도어 기계적 문제
  • 권한 문제

해결 방법:

  1. 슬라이딩 도어 현재 상태 확인
  2. 슬라이딩 도어 전원 상태 확인
  3. 수동으로 슬라이딩 도어 작동 확인
  4. 설정 → 사용자 관리에서 권한 확인
  5. 슬라이딩 도어 재부팅 (전원 재연결)

앱에서 슬라이딩 도어 상태가 업데이트되지 않습니다

가능한 원인:

  • 블루투스 연결 문제
  • 앱 캐시 문제
  • 슬라이딩 도어 펌웨어 문제

해결 방법:

  1. 앱 새로고침 (당겨서 새로고침)
  2. 앱 완전 종료 후 재실행
  3. 슬라이딩 도어 재연결
  4. 설정 → 앱 정보 → 캐시 삭제

📱 앱 문제

앱이 실행되지 않습니다

가능한 원인:

  • iOS 버전 호환성 문제
  • 앱 손상
  • 저장공간 부족

해결 방법:

  1. iOS 13.0 이상인지 확인
  2. 앱 삭제 후 재설치
  3. 기기 저장공간 확인
  4. 기기 재부팅

앱이 느리게 작동합니다

가능한 원인:

  • 기기 성능 문제
  • 앱 캐시 과다
  • 백그라운드 앱 과다

해결 방법:

  1. 백그라운드 앱 정리
  2. 설정 → 앱 정보 → 캐시 삭제
  3. 기기 재부팅
  4. 앱 재설치

🔐 권한 문제

블루투스 권한이 거부되었습니다

오류 메시지: "블루투스 권한이 필요합니다"

해결 방법:

  1. 설정 → SmartDoors → 블루투스 허용
  2. 앱 재실행
  3. 권한 요청 시 "허용" 선택

위치 권한이 거부되었습니다

참고: 블루투스 스캔을 위해 위치 권한이 필요합니다.

해결 방법:

  1. 설정 → SmartDoors → 위치 허용
  2. "앱 사용 중에만" 또는 "항상" 선택
  3. 앱 재실행

🔧 고급 문제 해결

슬라이딩 도어 진단 방법

진단 모드 실행:

  1. 앱에서 슬라이딩 도어 선택
  2. 설정 → 진단 실행
  3. 진단 결과 확인
  4. 문제 발견 시 제조사에 문의

앱 로그 확인

로그 확인 방법:

  1. 설정 → 개발자 옵션 → 로그 활성화
  2. 문제 상황 재현
  3. 설정 → 로그 내보내기
  4. 로그 파일을 지원팀에 전송
